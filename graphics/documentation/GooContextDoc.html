Class GooContext

  • java.lang.Object
    • graphicmotor.GooContext

  • public class GooContext
    A GooContext is the main class to control graphic motor's operations into a specific canvas created at the construction. It permits to link the design and artistic ressources from conceptors with logical and optimize rendering into its specified canvas.

      • setEntityColorMask

        public void setEntityColorMask(int entityReference, float r, float g, float b)
        Set the r g b color of the referenced entity

        if 0.0f 0.0f 0.0f are used for RGB colors arguments, the entity will be transparent ;
        if 1.0f 1.0f 1.0f are used, the entity will be render like in the original image, with a maximal opacity.

        Users can add them own alpha filters :
            float alpha = 0.8 ;
            setEntityColorMask(entityReference, r * alpha, g * alpha, b * alpha);
        where r, g, b type is float

        Parameters:
        entityReference - the referenced entity.
        r - the red value [ 0 ; 1 ].
        g - the green value [ 0 ; 1 ].
        b - the blue value [ 0 ; 1 ].

      • destroyEntity

        public void destroyEntity(int entityReference)
        Destroy the referenced entity, and free the specified memory space

        When an entity is destroy, the reference is free for another one.

        Users shall never call any function with the reference of an entity previously destroyed. In the best case, it won't change anything to the execution. In the worst one, another entity could have been created with that same reference, and some conflit will happen.

        Parameters:
        entityReference - the referenced entity.
